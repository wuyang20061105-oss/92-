# 考研分数线查询系统

985/211/热门双非高校工科专业考研分数线数据采集与可视化查询工具。

## 功能特点

- **18源数据采集**：从考研帮、研招网、各校官网等18个数据源爬取分数线数据
- **智能查询**：支持按学校、专业、年份、地区多维度筛选
- **数据对比**：可选择多所学校进行分数线对比分析
- **可视化图表**：使用 Chart.js 展示历年分数线趋势
- **覆盖全面**：涵盖985、211及热门双非院校，工科类专业分类

## 使用方法

### 1. 安装依赖

```bash
pip install requests beautifulsoup4
```

### 2. 运行爬虫

```bash
python scraper.py
```

运行后会在当前目录生成 `data.json` 数据文件。

### 3. 查看数据

直接在浏览器中打开 `index.html` 即可使用查询系统。

## 文件结构

```
├── index.html      # 前端查询界面 (Vue.js 3 + Chart.js)
├── scraper.py      # 18源数据爬虫
├── data.json       # 爬取的数据 (运行爬虫后生成)
├── data.js         # 前端数据文件
└── README.md       # 项目说明
```

## 技术栈

- **前端**：Vue.js 3、Chart.js
- **后端**：Python 3、requests、BeautifulSoup4
- **数据**：JSON 格式存储
